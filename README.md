## II. 使用技术

- 前端
  1. HTML、CSS、JavaScript；
  2. Ant Design Pro 项目模板 & ProComponents组件库；
  3. React开发框架；
  4. Umi框架与全局请求响应拦截器；
  5.  TypeScript；
  6. 正向/反向代理；

- 后端
  1. Java编程语言；Spring+SpringMVC+SpringBoot框架；
  2. MyBatis+MyBatis Plus框架+MyBatis-X工具；
  3. MySQL数据库；
  4. jUnit单元测试；
  5.  通用返回对象、自定义异常及错误码、全局异常处理器；
